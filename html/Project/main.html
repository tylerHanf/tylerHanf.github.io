Proposal


Idea - Procedural tree (or plant) generation via L-Systems using OpenGL compute shaders

The goal is to use OpenGL compute shaders to procedurally generate trees. My initial idea is to use L-Systems for the procedural generation. Since the challenge of this unknown, there are options to add such as including a GUI to alter generation parameters on the fly (probably using ImGui for the GUI library), enhance procedurally generated texture quality, and add scene real-time camera interactivity as needed.

Team Members - None
Project Components - OpenGL with GLEW and GLFW, possibly GLM for the math library (may use custom library), and ImGui for the GUI library if needed Timeline - Generate L-System and draw at least in 2D the generated trees (Mid-point review) Final Presentation - This will be a powerpoint with snippets and possible videos demoing the project
