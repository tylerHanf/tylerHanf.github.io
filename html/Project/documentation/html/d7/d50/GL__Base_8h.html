Detailed Description

This provides a basic interface for the core components of OpenGL This is designed to make working with OpenGL easier, not necessarily to maintain utmost efficiency or to enable utmost control by the user.

Author
Tyler Hanf

This is designed to be largely header-only for easy use in other programs when just wanting to interact with OpenGL
