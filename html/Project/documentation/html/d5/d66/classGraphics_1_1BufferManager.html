A high level function to add data to the buffer manager.

The manager attempts to add buffer data to an existing buffer of the same vertex attribute, or creates a new one if needed

Parameters
dataThe data to be added
handle[out] The handle that determines where the data is in the buffer manager
Returns
true If able to add data to the manager
